

# unicoder | ![logo](https://github.com/zainmk/unicoder/blob/main/icon48.png) 

#### purpose
unicoder is a chrome extension that allows for quick inline text replacement via unicode (hex) or custom aliases, prompted by "/" then hitting the 'Tab' key, within any input text field in browser.

#### stack
HTML, CSS, JS, CHROME EXTENSION (MANIFEST.JSON)
